# Calculadora-simples-JAVA
Calculadora simples em JAVA, com entrada de 2 números. Feito com o propósito de praticar switch case.
O programa inicia pedindo que digite os 2 números que serão usados na operação
Em seguida é mostrado 6 opções no qual o usuário pode escolher: 
1 - Adição 
2 - Subtração 
3 - Multiplicação 
4 - Divisão 
5 - Selecione outros números 
6 - Fechar programa
